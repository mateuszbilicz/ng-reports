How to use NG Reports

NG Reports provides you with Angular library that you can import in any Angular 18+ project. There're many configuration options that help you improve your experience as developer and user experience. On this page, we will go through all steps from how to install NG Reports, configure it and how to use it.

Installation

First, download NG Reports package .tgz file and put it inside project directory and use npm install command.

CMD 
npm i ./ng-reports.tgz

App configuration

Depending on your needs, you can enable few different collectors. All collectors gather information to report logs, which can be sent with report form.

  • NgReportsConsoleService - console errors collector
  • ngReportsHttpInterceptor - http requests collector
  • NgReportsService - contains collectors:
    • console.log
    • mouse clicks
    • route changes
    • form data

Adding collectors

Adding collectors, NG Reports service and configuration must be done inside app.config.ts file. If not, NG Reports wouldn't be able to work.

Usage

After you've added collectors and properly configured NG Reports - it's working in the background and already collecting logs. Now you can bind form data to the service or show report form dialog.

Display the dialog
